1. Stream (Factories)

الميثودالشرح
empty()Stream فارغ.
of(T value)عنصر واحد.
of(T... values)عدة عناصر.
generate(Supplier)مولّد لا نهائي (عادة مع limit).
iterate(seed, UnaryOperator)تكرار لا نهائي.
iterate(seed, hasNext, next)تكرار بشرط توقف (Java 9+).
builder()منشئ يدوي: add ثم build.
concat(a, b)وصل Stream.
ofNullable(T t)Stream بعنصر أو فارغ (Java 9+).

2. Collections / Arrays / IO / Regex

الميثودالشرح
Collection.stream()Stream متسلسل.
Collection.parallelStream()Stream موازي.
Arrays.stream(T[] array)من Array إلى Stream.
Arrays.stream(int[]/long[]/double[])من مصفوفة بدائية إلى Stream بدائي.
Files.lines(Path)Stream لسطر الملف (AutoCloseable).
Pattern.splitAsStream(text)Stream نصوص مقسومة بـ Regex.
