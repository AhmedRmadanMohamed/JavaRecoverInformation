Java Streams — Intermediate Operations

كل العمليات الوسيطة (Intermediate) في Stream, IntStream, LongStream, DoubleStream

1. Stream

الميثودالشرح
map(Function)تحويل كل عنصر لعنصر جديد.
mapToInt/Long/Doubleتحويل إلى Stream بدائي.
flatMap(Function)تفريد Streams داخلية.
flatMapToInt/Long/Doubleتفريد إلى Primitive Streams.
mapMulti(BiConsumer)بديل فعال لـ flatMap (Java 16+).
filter(Predicate)ترشيح العناصر.
distinct()إزالة التكرارات.
sorted()ترتيب طبيعي.
sorted(Comparator)ترتيب بمقارن مخصص.
peek(Consumer)مراقبة/تصحيح (debug).
limit(n)أخذ أول n عناصر.
skip(n)تخطي أول n عناصر.
takeWhile(Predicate)أخذ عناصر طالما الشرط صحيح (Java 9+).
dropWhile(Predicate)تجاهل طالما الشرط صحيح (Java 9+).
unordered()إلغاء ضمان الترتيب لتحسين الأداء.
onClose(Runnable)إجراء عند إغلاق الـ Stream.

2. IntStream / LongStream / DoubleStream

الميثودالشرح
map(XUnaryOperator)تحويل ضمن نفس النوع البدائي.
mapToObj(XFunction)تحويل إلى Stream مرجعي.
mapToInt/Long/Doubleتحويل لأنواع بدائية أخرى.
flatMap(XFunction)تفريد إلى Streams بدائية.
filter(XPredicate)ترشيح.
distinct(), sorted()إزالة التكرار/ترتيب.
peek(XConsumer)مراقبة.
limit(n), skip(n)قصّ.
takeWhile(XPredicate), dropWhile(XPredicate)Java 9+.
asDoubleStream()تحويل إلى DoubleStream.
boxed()تحويل إلى Stream من الكائنات (Integer/Long/Double).
