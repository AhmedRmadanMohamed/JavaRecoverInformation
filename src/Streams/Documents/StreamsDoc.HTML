Java Streams — الملخّص السريع (Cheat Sheet)

أربع مفاهيم أساسية تفهّمك الـ Streams من غير تعقيد: Pipeline · Lazy · One‑shot · Immutable

Pipeline

الـ Stream بيتقسم 3 مراحل: Source (المصدر) → Intermediate (العمليات) → Terminal (النتيجة).
Source → Intermediate → Terminal

Lazy

الـ Intermediate ما تتنفّذش غير لما تعمل Terminal — الزر اللي يشغّل الخط كله.

One‑shot

الـ Stream «يتاكل» مرة واحدة بس؛ لو عايز تكرّر، اعمل Stream جديد من المصدر.

Immutable

الـ Stream ما يغيّرش الداتا الأصلية؛ بيرجع لك نتيجة جديدة مبنية على العمليات اللي طلبتها.
جاهز للخطوة الجاية؟ جرّب تطبّق كل مفهوم على كود صغير، وبعدين زوّد التعقيد واحدة واحدة.
